== Source Listing With Callouts

[source,swift]
----
struct Greeter {
    let prefix: String

    init(prefix: String = "Hello") { <1>
        self.prefix = prefix          <2>
    }

    func greet(_ name: String) {      <3>
        print("\(prefix), \(name)!")  <4>
    }
}
----
<1> Demonstrates an initializer inside the listing.
<2> Uses a stored property assignment.
<3> Methods can also receive callouts.
<4> The final line references a callout anchor.
