# ``AsciiDocCore``

The core logic for parsing AsciiDoc documents.

## Overview

`AsciiDocCore` provides the fundamental data structures and parsing engine for processing AsciiDoc text. It parses source text into an Abstract Semantic Graph (ASG) or Intermediate Semantic Model (ISM) represented by `AdocDocument`.
